What do you want to see?

Just the Ticket is a React web app using Firebase that allows users to search for top movies currently playing in theatres, save favourites, and find showtimes near them. The application leverages three APIs: MovieDB, International Showtimes, and New York Times Movie Reviews.

This project was made in partnership with Robyn Dalgleish and Natasha Teng.

Infinity below the fold

This project leverages NewsAPI to aggregate headlines news articles from 5,000 news sources and blogs around the world. An infinite scroll loads more content before the user can reach the bottom of the page, and a drop-down allows to the user to filter the news by 52 different countries. Local weather provided by Open Weather Map Weather API.

Created in partnership with Quinn Anderson

Out of the pantry and onto the plate

I made this project in response to a problem that my husband and I were having - keeping track of what we had in the kitchen! Fridge to Table stores the contents of our fridge in Firebase, and warns us when food need to be eaten soon. It also holds a shared grocery list we can track from anywhere. This application was designed mobile-first for use on the go.
